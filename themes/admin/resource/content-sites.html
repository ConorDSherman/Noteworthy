Evernote Integration

Sandbox

The Evernote Sandbox allows you to test Noteworthy with a test Evernote account. This is useful if you'd like to see whether the software is a good match for what you want to accomplish, or if you just don't trust my software with your real Evernote account.

Production

Production refers to the real Evernote servers. By using the Production servers you will be able to share content from the notebooks you specify online.
